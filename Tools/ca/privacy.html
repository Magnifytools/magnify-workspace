Dades que recollim

Magnify Tools és una landing estàtica que no conté formularis ni recull dades personals directament. No utilitzem galetes pròpies ni de tercers.

El servei està allotjat a GitHub Pages, que per raons tècniques pot registrar logs d'accés anònims (adreça IP, navegador, data de visita) amb finalitat de seguretat i integritat del servei. Aquestes dades són gestionades per GitHub Inc.

Les eines concretes — Mirror i Accent — disponibles en subdominis independents, tenen les seves pròpies polítiques de privacitat:

  • Mirror: mirror.magnifytools.com/privacy
  • Accent: accent.magnifytools.com/privacy
03

Finalitat i base legal

Les dades tècniques registrades pel hosting es tracten sobre la base de l'interès legítim (art. 6.1.f RGPD) per garantir la seguretat, el rendiment i la integritat del servei.

L'usuari pot exercir els drets d'accés, rectificació, supressió, oposició, limitació i portabilitat escrivint a hola@magnify.ing.
